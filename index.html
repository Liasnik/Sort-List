кнопка ALL KEYWORDS TWO AREAS: выводит в поле "Данные с ключевым словом" данные, содержащие ключевые значения. И одновременно выводит данные без ключевых значений в поле "Данные без ключевого слова".
Чтобы вывести список включающий только уникальные строки (исключить дубликаты) БЕЗ ДОПОЛНИТЕЛЬНЫХ ИЗМЕНЕНИЙ этого текста, нужно в первое поле для ввода ключевых слов ввести набор символов который точно нигде в тексте не встречается, например несколько звёздочек и т.п.

кнопка STRICT KEY TWO AREAS: добавляет в поле "Данные с ключевым словом" данные, включающие точные значения конкретных отдельных ячеек таблицы, записанных в ячейке полностью либо по последним символам этих значений. И одновременно вставляет список без этих данных в поле "Данные без ключевого слова". Для такой же сортировки по крайнему правому столбцу нужно копировать данные из таблицы вместе со следующим пустым столбцом.

кнопка WITH KEYWORDS: вставляет данные содержащие все ключи в поле "Данные с ключевым словом".

WITHOUT KEYWORDS: выводит список без данных, содержащих проверочные ключи в поле "Данные без ключевого слова".

Все кнопки выводят только уникальные строки.
