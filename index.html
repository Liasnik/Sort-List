Данные без ключевого слова:

ONE KEYWORD TWO AREAS: добавляет в поле "Данные с ключевым словом" данные, включающие точные значения конкретных отдельных ячеек таблицы Excel по полному значению, записаному в ячейке либо по окончанию этих значений. И одновременно вставляет входящий список без этих данных в поле "Данные без ключевого слова". Для такой же сортировки по крайнему правому столбцу нужно копировать данные из таблицы вместе со следующим пустым столбцом.
При каждом нажатии этой кнопки данные в поле "Данные с ключевым словом" плюсуются. В поле "Данные без ключевого слова" - перезаписываются.

ALL WITH KEYWORDS: вставляет данные содержащие все ключи в поле "Данные с ключевым словом"

ONE WITHOUT KEYWORD: выводит список без данных, содержащих один проверочный ключ в поле "Данные без ключевого слова". (Эта функция временно недоступна. Работает как предыдущая кнопка).

Все кнопки выводят только уникальные строки, исключая дубликаты, имеющие полное совпадение по всей строке.
С незаполненными полями для ввода ключей выводится просто весь список, исключая дубликаты.
